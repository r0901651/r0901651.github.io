Omschrijving

Tijdens mijn studiecarrière aan Thomas More campus Geel (2023) heb ik verschillende projecten gemaakt. Eén daarvan behoorde onder de study module Windows Deployment. In dit project moesten wij in groepen van twee een project maken. Samen met mijn medestudent Brett Meykens hadden wij een case waar we 2000 oude toestellen in het bedrijf moesten vervangen door nieuwere toestellen. Dit maakte dat we Software inventarisatie moesten doen van de oude clients, OS captures uitvoerden, Custom software deployments en ook OS deployments hebben moeten uitvoeren op de nieuwe toestellen. Hiervoor werdt gebruikt gemaakt van SCCM en een windows deployment server via VMWare.


De taakverdeling was als volgt:

  • Anthony Van Roy :
  • Operating System Preparation
  • Operating System Capture
  • Task Sequence
  • PXE Boot

  • Brett Meykens :
  • Software Inventory Op De Oude Clients
  • Software Deployment Voor Common Software
  • Software Deployment Per User
  • Client Import
